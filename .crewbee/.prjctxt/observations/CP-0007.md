# CP-0007 Remove Migration Feature

## Summary

Removed directory migration as a feature. `.crewbee/` is a naming requirement and the only production context directory supported by the scaffold.

## Key Changes

- Deleted migration API/CLI implementation.
- Removed migration tests and public API declarations.
- Updated README, Chinese guide, and project context records to describe `.crewbee/` as the required directory name rather than a migration target.

## Verification

- `npm run diagnostics` passed.
- `npm run typecheck` passed.
- `npm test` passed with 10 tests.
- `npm run build` passed.

## Follow-ups

- Keep initialization focused on creating `.crewbee/` directly.
