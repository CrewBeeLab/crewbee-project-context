# CP-0002 Template Directory Clarification

## Summary

Clarified the distinction between scaffold source templates and production project context directories.

## Key Changes

- Renamed scaffold source template directory from `templates/crewbee/` to `templates/crewbee-template/`.
- Documented that `.crewbee/` is the production/runtime context directory name.
- Added decision and memory entries for this rule.

## Verification

- `npm run diagnostics` passed.
- `npm run typecheck` passed.
- `npm test` passed with 3 tests.
- `npm run build` passed.

## Follow-ups
